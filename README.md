# Usage

Before running the code, you will need to register in CDS Copernicus [website](https://cds.climate.copernicus.eu/user/register?destination=%2F%23!%2Fhome) to be able to access climatic data.

Then, you need to configure the api following steps shown [here](https://cds.climate.copernicus.eu/api-how-to)

These steps are needed to run the script api_cds.py and download the climatic data in .nc format required to follow the main.R script.
